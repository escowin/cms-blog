## Installation
Terminal instructions

  1. Navigate to desired download directory
  2. Run the following commands:

      ```
      $ git clone git@github.com:escowin/fitness-logbook.git; 
      ```
      ```
      $ cd fintess-logbook;
      ```
      ```
      $ npm i; 
      ```
      ```
      $ npm run seed;
      ```
      ```
      $ npm start;
      ```
  3. Open http://localhost:3001 in browser.
